Unblocked Ledger Coin


The first linearly scalable, self-governing payment network

Unblocked Ledger Coin is a payment network designed to remedy the problems of traditional blockchains — scalability, decentralization, security and long term sustainability. The technology being developed for ULC can accommodate billions of daily active users and can be applied to many future global-scale decentralized applications. The ULC payment network is the first application of this technology.

Technology

The Unblocked Ledger project is building a peer-to-peer payment network that incorporates sharding and auto-scaling to provide high throughput, low latency, and immediate finality while maintaining the highest level of decentralization and security.


The supporting technology that allows for these features is our original Unblocked Consensus Algorithm, which is based on Proof-of-Quorum, in combination with our Unblocked Sharded Ledger system, which is a distributed ledger that addresses the problems of linear scaling and state sharding as the network grows.

Features

Fast

Transactions are processed immediately and not grouped into blocks resulting in fully confirmed transaction times of just a few seconds.

Scalable Processing

Transactions are routed to groups of nodes (shards) so that not all nodes have to process all transactions and the network can scale to millions of transactions per second.

Scalable Storage

State data is distributed across multiple shards resulting in networks that can hold trillions of accounts without requiring each node to provide petabytes of storage.

Truly Decentralized

Off the shelf computers can be used as full nodes to earn a steady regular income without joining mining pools and providing the highest level of decentralization.

Secure

Proof-of-Quorum consensus provides the highest level of security and data consistency while maintaining energy efficiency.

Sustainable

A community-activated soft fork system allows setting network parameters and provides a maintenance fund.

Token

During the development phase of the ULC network, we will be issuing an ERC-20 token (Unblocked Ledger Token) that will be given to developers who complete bounties for the project. To incentivize the development of the ULC network software, a max supply of 1,000,000,000 (one billion) Unblocked Ledger Tokens (ULT) will be created in an ERC-20 contract. The tokens will only be distributed as bounties to help create and launch the ULC peer-to-peer network software. Tokens will not be sold.

What is the purpose of the Unblocked Ledger project?

The Unblocked Ledger project aims to build and showcase a highly scalable, truly decentralized payment network referred to as Unblocked Ledger Coin (ULC). The software built to showcase this payment network can be reused to build other networks for enterprise or public use.

What problem does the UL project solve?

Unblocked Ledger aims to solve the scalability problem of traditional blockchains and provide a secure, fast, and sustainable payment network for enterprise and public use.

What are ULT?

Unblocked Ledger Tokens (ULT) are ERC-20 compliant tokens on the Ethereum blockchain. ULT will be given for bounties completed to create and launch the Unblocked Ledger Coin network.

What is the difference between ULT and ULC?

ULT are ERC-20 tokens on the Ethereum blockchain, while ULC are coins on the peer-to-peer network created by the Unblocked Ledger project.

Why is there no ICO for ULT?

We are not doing an ICO because it may violate securities and exchange regulations in some jurisdictions. Also we want the direct contributors to the project to benefit as opposed to speculators.


ULT will only be distributed as bounties to those who help with developing and launching the project. Only those who believe in the project enough to directly contribute their effort to it will earn ULT. This is very similar to coins like bitcoin being earned by miners contributing resources to the network.


We will also do some airdrops to grow the community and give them a chance to test out the ease and speed of our payment network.

What is the purpose of ULC?

The main purpose of coins on the ULC network is to provide trustless value transfer irrespective of location. In addition, the ULC network aims to provide very high throughput, low latency, immediate finality and very low transaction fees.

What is Proof-of-Quorum?

Proof-of-Quorum is a type of consensus algorithm which provides a provable receipt for every accepted transaction. It is energy efficient and does not require staking coins.

Why does ULC not use a blockchain?

Grouping transactions into blocks has many inherent disadvantages. Processing each transaction individually provides the highest possible throughput and lowest latency.

What are the disadvantages of blockchains?


Throughput

The transaction throughput rate becomes limited by the average time between blocks and the maximum allowed size of the block. Adding more computers to the network does not help to increase the throughput; rather it has the negative effect of increasing the network bandwidth consumed by each transaction.

Node Bias

A single node (the one which produces the block) gets to decide which transactions will be in the block. That node can be biased and choose not to include some transactions or even create blocks with no transactions other than it’s own.

Time Resolution

The transactions in a block are considered to have occured at the same time. For some applications such as exchanges or games, this does not provide sufficient time resolution.

Is the UL project open source?

Once the project is completed, the source code will be made available under a Creative Commons license.
