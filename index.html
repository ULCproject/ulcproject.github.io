Unblocked Ledger Coin


The first linearly scalable, self-governing payment network

Unblocked Ledger Coin (ULC) is a payment network designed to remedy the problems of traditional blockchains — scalability, decentralization, security and long term sustainability. The technology being developed for ULC can accommodate billions of daily active users and can be applied to many future global-scale decentralized applications. The ULC payment network is the first application of this technology.

Technology

The ULC Project is building is building a peer-to-peer payment network that incorporates sharding and auto-scaling to provide high throughput, low latency, and immediate finality while maintaining the highest level of decentralization and security.


The supporting technology that allows for these features is our original Unblocked Consensus Algorithm, which is based on Proof-of-Quorum, in combination with our Unblocked Sharded Ledger system, which is a distributed ledger that addresses the problems of linear scaling and state sharding as the network grows.

Features

Fast

Transactions are processed immediately and not grouped into blocks resulting in fully confirmed transaction times of just a few seconds.

Scalable Processing

Transactions are routed to groups of nodes (shards) so that not all nodes have to process all transactions and the network can scale to millions of transactions per second.

Scalable Storage

State data is distributed across multiple shards resulting in networks that can hold trillions of accounts without requiring each node to provide petabytes of storage.

Truly Decentralized

Off the shelf computers can be used as full nodes to earn a steady regular income without joining mining pools and providing the highest level of decentralization.

Secure

Proof-of-Quorum consensus provides the highest level of security and data consistency while maintaining energy efficiency.

Sustainable

A community-activated soft fork system allows setting network parameters and provides a maintenance fund to ensure the network can support itself and remains flexible.

Token

During the development phase of the ULC network, we will be issuing an ERC-20 token (Unblocked Ledger Token) that will be given to developers who complete bounties for the project. To incentivize the development of the ULC network software, a max supply of 1,000,000,000 (one billion) Unblocked Ledger Tokens (ULT) will be created in an ERC-20 contract. The tokens will only be distributed as bounties to help create and launch the ULC peer-to-peer network software. Tokens will not be sold.

What is the purpose of the ULC Project?

The ULC Project aims to build and showcase a linearly scalable, self-governing, truly decentralized payment network with its own coin called ULC. The software built to showcase this payment network provides a platform that can be reused to build other networks for enterprise or public use.

Does this project have a token?

Yes, we are distributing an ERC-20 token on the Ethereum network called Unblocked Ledger Token (ULT). The token can later be transferred to the network developed by this project.

What is the difference between ULT and ULC?

ULT are ERC-20 tokens on the Ethereum blockchain, while ULC are coins on the peer-to-peer network created by this project. Once the software for this project is complete and the main net launched, ULT can be transferred to the network and become ULC.

Why is there no ICO?

Instead of selling our tokens for other tokens (like Bitcoin or Ethereum) and using that to pay the developers, we are directly giving our tokens as gratuity to developers for helping to create the ULC software. We are also giving our tokens for completing bounties or providing other services to the project. The tokens will gain value based on the efforts of those who contributed to the project and the contributors will be the first to realize the gains of their efforts. This better aligns the incentives to ensure the project is successfully completed.

How can I obtain tokens?

There are several ways. If you are a developer you should fill out the Join form and request to join the development team and earn tokens that way. We also post bounties from time to time and you can earn tokens by completing them. We will have a faucet in the future, so signup to be notified about it. You can contact the developers and buy tokens directly from them. You can also check your favorite exchange to see if they have chosen to list our token.

Is this project similar to EOS?

In some ways it is similar to EOS since we are both building platform software and are distributing a token that can later be transferred to the network developed by the project. However, our network is technically very different in many ways. One key difference is our goal of maximum decentralization with many nodes able to join the network and low resource requirements for the nodes.

Is the ULC Project open source?

Not during development. Once the project is completed, the source code will be made available under a Creative Commons BY-NC-SA license with a restriction on commercial use. After 10 years, it will be available under a Creative Commons BY license with no restrictions. Anyone interested in reviewing the code during development, may do so with a NDA which expires after the code is released.
